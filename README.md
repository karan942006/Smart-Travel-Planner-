# ✈️ About The Project

**Smart Travel Planner Chatbot** is a simple yet interactive travel assistant developed using **HTML, CSS, and JavaScript**.

The chatbot helps travelers quickly access information about popular destinations including:

🌍 Tourist Attractions
🏨 Hotels
☁️ Weather Information
💰 Travel Budget Estimates
🗓️ Travel Itineraries

Users can simply type a query and instantly receive travel recommendations.

### Step 1

User enters a travel query.

### Step 2

JavaScript analyzes the text.

### Step 3

The chatbot identifies:

* City
* Query Type

### Step 4

Relevant information is displayed instantly.
